Italy MIR Helper Privacy Policy

Effective Date: May 21, 2026

Italy MIR Helper is a Chrome extension intended for authorized internal business use. The extension assists users with transferring Italy Manufacturer Incident Report XML files from SAP CRM to the Italian SISN Manufacturer Incident Report portal.

Information the Extension Handles

The extension may read or interact with the following information only when the user is using the supported SAP CRM and SISN MIR websites:

- SAP CRM email page content needed to determine whether the BCC field matches the Italy MIR reporting address.
- XML attachment information needed to identify and download the relevant MIR XML file.
- The downloaded XML file selected by the user-directed workflow.
- SISN MIR page content and form controls needed to select the correct report option, upload the XML file, and advance through the MIR workflow.
- Temporary workflow state stored in the browser, such as whether the user started the workflow and the most recent XML file selected for upload.

This information may include personally identifiable information, health information, personal communications, and website content contained in the CRM email, XML file, or SISN MIR form.

How the Information Is Used

The extension uses this information only to provide its single purpose: assisting authorized users with preparing an Italy MIR submission by moving a CRM XML attachment into the SISN MIR workflow.

The extension does not use this information for advertising, analytics, profiling, creditworthiness, lending, or any unrelated purpose.

Information Sharing

The extension does not sell user data.

The extension does not transfer user data to the developer or to third-party services. The only user-directed transfer is the upload of the selected XML file to the Italian SISN MIR portal as part of the MIR submission workflow.

Data Storage and Retention

The extension stores only temporary workflow state in the user’s browser. This may include the current automation step and the latest downloaded XML filename. This data is used only to resume or complete the active MIR workflow and is not sent to the developer.

Security

The extension runs only on the approved SAP CRM stage site and the Italian SISN MIR portal. It does not run on unrelated websites. Data is handled locally in the browser except when the user-directed workflow uploads the XML file to the SISN MIR portal over HTTPS.

User Control

Users initiate the workflow by clicking the extension’s helper button. The extension stops before final report submission and does not submit a MIR report without user review.

Changes to This Policy

This privacy policy may be updated if the extension’s functionality changes. Updates will be reflected by changing the effective date above.

Contact

For questions about this privacy policy or the extension, contact the extension publisher.
